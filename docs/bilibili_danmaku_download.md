# B站弹幕下载功能说明

## 功能位置
设置 > 弹幕下载 > 哔哩哔哩弹幕下载

## 使用步骤

### 1. 首次使用 - 选择保存文件夹
- 点击"哔哩哔哩弹幕下载"
- 系统会提示选择弹幕保存文件夹
- 点击"选择文件夹"，选择你想保存弹幕的位置
- 文件夹路径会被保存，下次使用时无需重新选择

### 2. 选择下载模式
在文件夹选择完成后,会显示下载模式选项:
- **整季模式**(默认): 自动下载整季所有集数,跳过已存在的文件
- **单集模式**: 只下载当前输入链接对应的集数

### 3. 输入视频链接
- 打开B站，找到想下载弹幕的视频或番剧
- 复制视频链接（如：https://www.bilibili.com/video/BV1xx411c7mD）
- 或复制番剧链接（如：https://www.bilibili.com/bangumi/play/ss12345 或 ep123456）
- **支持直接粘贴分享文本**（如："我用哔哩哔哩向你分享了视频...链接：xxxxx"）
- 粘贴到输入框中
- 点击"开始下载"

**番剧下载说明**：
- **整季模式**: 输入番剧任意一集的链接,会**自动下载整季所有集数**的弹幕
  - 自动跳过已存在的文件,避免重复下载
  - 显示实时下载进度和统计
  - 例如输入"轻音少女第二季第1集"链接,会下载第二季全部集数
- **单集模式**: 只下载输入链接对应的那一集弹幕

### 3. 下载过程
- 出现旋转的加载动画,表示正在下载
- **整季模式**会显示:
  - 当前下载进度(第X/总数集)
  - 当前集的标题
  - 成功/失败统计
- 下载过程中请勿关闭对话框
- 完成后会显示Toast提示

### 4. 更改保存位置
- 在输入链接界面点击"更改保存文件夹"
- 重新选择新的保存位置

## 支持的链接格式
### 普通视频
- BV号链接：https://www.bilibili.com/video/BV1xx411c7mD
- AV号链接：https://www.bilibili.com/video/av170001
- 移动端分享链接也支持
- **支持从分享文本中自动提取链接**

### 番剧/影视
- 季度链接：https://www.bilibili.com/bangumi/play/ss12345
- 单集链接：https://www.bilibili.com/bangumi/play/ep123456
- 支持国创、番剧、电影、纪录片、电视剧、综艺等所有PGC内容
- **输入任意一集链接，自动下载整季所有集数弹幕**

## 文件命名
弹幕文件会以视频/番剧标题命名，自动保存为 `.xml` 格式
例如：
- 普通视频：`【东方】Bad Apple!! PV【影絵】.xml`
- 番剧单集：`某科学的超电磁炮 - 第1话.xml`
- 番剧整季：会生成多个文件，如 `轻音少女第二季 - 第1话.xml`、`轻音少女第二季 - 第2话.xml` 等

## 注意事项
1. 需要网络连接
2. 确保已授予存储权限
3. 建议在WiFi环境下使用
4. 非法字符会被替换为下划线

- 无需登录
- 无需认证
- 符合B站使用规范
