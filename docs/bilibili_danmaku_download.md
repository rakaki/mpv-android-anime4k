# B站弹幕下载功能说明

## 功能位置
设置 > 弹幕下载 > 哔哩哔哩弹幕下载

## 使用步骤

### 1. 首次使用 - 选择保存文件夹
- 点击"哔哩哔哩弹幕下载"
- 系统会提示选择弹幕保存文件夹
- 点击"选择文件夹"，选择你想保存弹幕的位置
- 文件夹路径会被保存，下次使用时无需重新选择

### 2. 输入视频链接
- 打开B站，找到想下载弹幕的视频
- 复制视频链接（如：https://www.bilibili.com/video/BV1xx411c7mD）
- 粘贴到输入框中
- 点击"开始下载"

### 3. 下载过程
- 出现旋转的加载动画，表示正在下载
- 下载过程中请勿关闭对话框
- 完成后会显示Toast提示

### 4. 更改保存位置
- 在输入链接界面点击"更改保存文件夹"
- 重新选择新的保存位置

## 支持的链接格式
- BV号链接：https://www.bilibili.com/video/BV1xx411c7mD
- AV号链接：https://www.bilibili.com/video/av170001
- 移动端分享链接也支持

## 文件命名
弹幕文件会以视频标题命名，自动保存为 `.xml` 格式
例如：`【东方】Bad Apple!! PV【影絵】.xml`

## 注意事项
1. 需要网络连接
2. 确保已授予存储权限
3. 建议在WiFi环境下使用
4. 非法字符会被替换为下划线

## 技术说明
- 使用官方B站API
- 无需登录
- 无需认证
- 符合B站使用规范
